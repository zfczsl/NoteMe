## 每周洞察指标重构

- 将「复盘次数」指标替换为「最高产日」（显示周几 · XX字）
- 将「记录天数」改为「活跃天数」，并加上 `/7` 后缀以直观显示占比
- 统计卡片顺序：记录条数 / 活跃天数 / 累计字数 / 最高产日

## AI Prompt 结构化数据修复

- 原来 buildDailyPrompt 没有要求 AI 返回结构化 JSON，导致情绪走势图无法获得真实 AI 数据
- 在 prompt 末尾追加了强制输出 JSON 块的指令，字段：energyLevel（1-10）、emotionScore（-1.0~1.0）、keywords（2-5个词）
- 在 formatSummaryContent 里过滤掉 JSON 块，用户看到的文字内容不含原始 JSON
- showCachedSummary 里也改为用 formatSummaryContent 而非 escapeHtml 直接展示

## 情绪默认值修复

- 原来 energyLevel 没有数据时默认0，导致图表空白
- 改为默认5分（一般），fallback 逻辑：如果 AI 有数据用 AI，否则用复盘情绪，否则用关键词分析，否则默认5
